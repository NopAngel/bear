# BearOS 🐻

A minimalist kernel project written from scratch for learning and experimentation.

## 📦 Features
- Basic kernel functionalities
- Custom build system
- Lightweight design

## 🛠️ Build & Run
```bash
git clone https://github.com/NopAngel/bear && cd bear
make compile && make iso
```
